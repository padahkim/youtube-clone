# Youtube mini project


- [x] Showing hot clip 25 list on root page 
- [x] Search keyword + enter/search button = change hot clip list to search results clip 
- [x] Click Play button will start the clip 

---

### **💡 What i used**

- React hook
- Youtube search API
- fetch, axios API
- Dependency Injection
- Post css
